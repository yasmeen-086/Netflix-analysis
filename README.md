# Netflix-analysis

## Overview

This project focuses on analyzing Netflix dataset to extract meaningful insights about content trends, genres, release patterns, and ratings. The analysis is performed using Python and data visualization libraries.

---

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Visualization of trends (movies vs TV shows, genres, release years)
* Insights into Netflix content distribution

---

## Tech Stack

* Python 
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

##  Project Structure

```
Netflix-analysis/
│── netflix_analysis.py      # Main analysis script
│── visualizations.py        # Visualization functions
│── README.md                # Project documentation
│── requirements.txt         # Dependencies
```

---

##  Key Insights

* Majority of content on Netflix is Movies
* Significant growth in content after 2015
* Drama and Comedy are the most common genres
* USA contributes the highest number of titles

---

## How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/Netflix-analysis.git
```

2. Navigate to project folder:

```
cd Netflix-analysis
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the script:

```
python netflix_analysis.py
```
